# whcc-cli-releases

Release binaries for the [whcc CLI](https://github.com/whcc/whcc-cli).

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/whcc/whcc-cli-releases/main/install.sh | sh
```

Or download a binary from the [latest release](https://github.com/whcc/whcc-cli-releases/releases/latest).
